# jack-of-all-trades-checker
League of Legends「何でも屋」スタック確認ツール
Jack of All Trades Checker

必要環境
- Python 3.13
- Git

起動方法

python -m venv .venv

.\.venv\Scripts\Activate.ps1

pip install -r requirements.txt

python run.py
